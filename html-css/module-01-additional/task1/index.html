Строчные элементы

  • a - Внутрь тега помещается текст, который будет отображаться на веб-странице
  • span - Предназначен для определения строчного элемента внутри документа
  • img - Предназначен для отображения на веб-странице изображений в различных графических форматах

Блочные элементы

  • h1...h6 - Эта группа тегов определяет текстовые заголовки разного уровня, которые показывают относительную важность секции, расположенной после заголовка
  • p - Разбивает текст на отдельные абзацы, отделяя друг от друга пустой строкой
  • ul, ol, dt - Теги для создания списков
  • div - Фактически используется для группировки тегов по значению контента
